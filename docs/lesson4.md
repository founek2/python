# 4. Lekce

### Cvičení - TODO aplikace
Napište konzolovou aplikaci, která si bude udržovat v poli text úkolu a jeho stav (hotovo/vytvořeno). Uživateli nabídne jednoduché menu:
```
Menu: (pro výběr stiskněte danou klávesu)
Zobrazi vše - 'a'
Zobrazit hotové - 'd'
Zobrazit nové - 'n'
Přidat úkol - 'c'
Ukončit - 'q'
```
